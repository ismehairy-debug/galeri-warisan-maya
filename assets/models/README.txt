KERIS 3D MODEL — one manual step
================================

The KERIS & ANYAMAN station automatically loads a real 3D keris from:

    assets/models/keris.glb

Model: "Keris" by apasipepgapuak on Sketchfab (CC Attribution, downloadable)
https://sketchfab.com/3d-models/keris-e66c540f891b426689dbd45fabb1f0a1

How to get the file (needs a free Sketchfab login — download can't be
scripted):

1. Open the link above and sign in to Sketchfab.
2. Click "Download 3D Model" and choose the glTF format (.glb if offered;
   if you get a .zip with scene.gltf + textures, ask Claude to convert it).
3. Save/rename the file to keris.glb and place it in this folder.
4. Commit and push — the station picks it up automatically and hides the
   photo plaque. Until then, the photo stays as fallback.

The CC Attribution credit line is appended to the station's description
automatically when the model loads.
